# DOIT - IELTS Practice Platform

## Project Structure

```
├── BE/     # Backend - Spring Boot
└── FE/     # Frontend - Vue 3
```

## Branches

- `main` - Production (stable releases)
- `DEV` - Development/Staging
- `BE` - Backend development
- `FE` - Frontend development

## Tech Stack

### Backend

- Java 17 + Spring Boot 3
- PostgreSQL + Redis
- Spring Security + JWT

### Frontend

- Vue 3 + TypeScript
- Pinia + Vue Router
- Tailwind CSS
